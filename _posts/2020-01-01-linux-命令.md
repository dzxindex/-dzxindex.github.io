# linux查看端口port所运行相关文件位置

```

1. 查看进程占用哪些文件的
 lsof -i:8080

2.使用cd打开pid所在的文件路径
  cd  /proc/{上一步查询出的pid号} 

3. 获取该进程所在目录
pwdx {上一步查询出的pid号}


# 使用netstat来查看端口对应的的pid
netstat -antlp|grep 9002
# 查看某个文件被哪个进程占用的
fuser -m -v /data/
```
